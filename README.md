Overview
This project analyzes retail sales data from a Superstore dataset using Excel.
It explores sales performance, customer behavior, pricing strategies, and profitability through charts, lookup functions, and conclusions.
Dataset
Columns: OrderID, OrderDate, CustomerName, Region, Category, ProductName, Quantity, UnitPrice, Discount%, PaymentMethod, Sales
Project Structure
Introduction – Overview & objectives
Charts – Sales trends & visual insights
Lookup Analysis – VLOOKUP, INDEX+MATCH, XLOOKUP, formulas
Conclusion – Key findings & recommendations
Tools
Microsoft Excel (Charts, PivotTables, Lookup Functions)
